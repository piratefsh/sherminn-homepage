Web Development

Newsletter Builder

Problem: It's easy to design newsletter templates. The hard part is teaching users how to edit it. Even with a GUI editor, newsletter templates get messed up when users add stories.

Solution: A simple form-based newsletter builder. The template is set with fixed layouts for each 'story' in the newsletter. Users can set editable fields such as the title and date of issue as well as add as many 'stories' as they need to.

Tools: HTML5, CSS3, Javascript, JQuery

Problem: I needed a way to hide personal or sensitive information when working on websites in public.

Tuter: The Ultimate Tutor Finder

// Currently under development

Problem: There are many academic tutoring resources at the University of Wisconsin-Madison. Unfortunately, they are very decentralized and it is hard for students to search for tutors.

Solution: A centralized database of tutors where students can search for tutors based on courses and locations.

Front-end: HTML5, CSS3, SASS, Javascript, JQuery.
Back-end: Ruby on Rails.

Currently working in a group of 5. I'm working on the front-end.
